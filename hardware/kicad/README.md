# KiCad Hardware

Save the KiCad project files for the industrial CAN sensor node in this folder.

Planned schematic sections:

1. 18–30 V input protection
2. 24 V to 5 V buck converter
3. 5 V to 3.3 V regulator
4. STM32 microcontroller
5. CAN transceiver and termination
6. 0–10 V analog input
7. digital sensor input
8. safe-state output
9. debug header, LEDs, and test points
